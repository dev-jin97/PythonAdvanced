# List Handling
> I think dealing with lists is the most important thing in Python coding.

## List method
```python
# append

fruits = ["apple", "orange"]
fruits.append("grape")

# List in list append
fruits.append(["grape", "mango"])

# pop
fruits.pop() # delete a mango

# use a index to pop
fruits.pop(0)

# remove
fruits.remove("orange")

# find of index
fruits.index("orange")

# count
fruits.count("apple")

# clear all list data
fruits.clear()

# sort
fruits.sort()
```
