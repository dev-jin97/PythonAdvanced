# List_Implication(리스트 내포)

## What is a List Implication?(_리스트 내포란?_)
- for문, if문 등을 지정하여 리스트를 간편하게 만드는 방법

```python
nums = [i for i in range(5)]

nums = [i for i in range(10) if i % 2 == 0]
```
